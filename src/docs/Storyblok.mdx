# Generating types for Storyblok 

## Generate the types

This will generate the types for the project from the most recent content in Storyblok.

```bash
  npm run storyblok:types
```

## Under the hood

A guide can be found [here](https://www.storyblok.com/faq/how-can-i-utilize-typescript-in-my-storyblok-project)
